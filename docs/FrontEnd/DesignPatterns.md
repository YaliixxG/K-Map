# <font color="#FBA500">Javascript 设计模式 </font>    

> 为了解决在开发中可能遇到的需求（相似），而提出的一套解决方法。

### 为什么要使用设计模式？  
使用设计模式的根本目的是为了减少项目变化所造成的影响。   

### 书籍  
《设计模式》、《大话设计模式》、《23种常见的设计模式》  

### 常用的设计模式  
* 工厂设计模式
* 单例设计模式
* 命令模式
* 模板模式
* 策略模式
* 观察者模式
* 命名空间模式

### 目的  
* 高内聚，低耦合
* 提高重用性，减少代码冗余
* 扩展性
* 稳定性  
......
